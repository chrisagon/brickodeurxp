Parcours utilisateurs — Section 3/4

Les 4 flux principaux de l'application

👦 Inscription d'un jeune
S'inscrit (email + mdp) → Saisit email parent → Parent reçoit magic link → Parent crée son compte → Liaison établie ✓
🏅 Demande de badge (jeune)
Choisit une compétence → Upload photo ou vidéo → Soumet la demande → Animateur notifié → Valide en séance → Badge attribué ✓

En cas de refus → commentaire de l'animateur visible par le jeune + possibilité de resoumettre

👪 Suivi parent
Reçoit magic link → Crée son compte → Voit le passeport enfant → Badges Brick + Codeur → Niveau ceinture actuel

Le parent reçoit une notification email à chaque nouveau badge validé

👨‍🏫 Validation animateur
Voit les demandes en attente → Consulte la preuve (photo/vidéo) →
Valide ✓ ou Refuse avec commentaire

Ces flux te semblent complets ? Des cas manquants ?
